TubeCompass サポート

TubeCompassの使い方やトラブルについては、以下をご確認ください。解決しない場合はお問い合わせください。

よくある質問

  • APIキーはどこで手に入りますか？
    アプリ内の「⚙ 設定」画面から「APIキーの取得方法を詳しく見る」を開くと、Google Cloud ConsoleでYouTube Data API v3のキーを取得する手順を確認できます。無料で取得でき、1日あたり約10,000ユニットまで利用できます。
  • 「APIキーが正しくない可能性があります」と表示されます
    Google Cloud Console側で「YouTube Data API v3」が有効化されているか、コピーしたキーに余分な空白が含まれていないかをご確認ください。
  • TubeCompass Proと無料版の違いは何ですか？
    詳細なキーワード指定・説明欄のテキストでの絞り込み・隠れた名作を探すフィルタ・言語フィルタ・広告非表示が、月額サブスクリプション「TubeCompass Pro」で利用できます。それ以外の検索・並び替え・お気に入り・ブロック機能などは無料でお使いいただけます。
  • サブスクリプションの解約・購入の復元はどうすればよいですか？
    解約はiPhone/iPadの「設定」→ご自身の名前 →「サブスクリプション」から行えます。機種変更などで購入内容が反映されない場合は、アプリ内「⚙ 設定」の「購入を復元」をお試しください。
  • 入力したAPIキーや検索条件はどこに保存されますか？
    すべてお使いの端末内にのみ保存され、開発者を含む外部には送信されません。詳しくはプライバシーポリシーをご確認ください。

TubeCompass Support

For help using TubeCompass, please check the FAQ below. If it doesn't resolve your issue, contact us.

Frequently Asked Questions

  • Where do I get an API key?
    Open the Settings screen (⚙) in the app and tap "View detailed instructions for getting an API key" for step-by-step guidance on obtaining a free YouTube Data API v3 key from Google Cloud Console (up to ~10,000 units/day at no cost).
  • I see "This API key may be invalid"
    Check that "YouTube Data API v3" is enabled for your key in Google Cloud Console, and that the copied key has no extra spaces.
  • What's the difference between Free and TubeCompass Pro?
    Detailed keyword targeting, description-text filtering, the "hidden gems" filter, the language filter, and ad-free browsing require the monthly TubeCompass Pro subscription. Everything else (search, sorting, favorites, blocking) is free.
  • How do I cancel my subscription or restore a purchase?
    Cancel anytime via iPhone/iPad Settings → your name → Subscriptions. If a purchase isn't reflected after switching devices, try "Restore Purchase" on the Settings screen in the app.
  • Where is my API key and search data stored?
    Everything is stored only on your device and is never sent to the developer or anyone else. See our Privacy Policy for details.
